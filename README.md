## Installation

Use the package manager [pnpm](https://pnpm.io/installation) for project dependencies.

```bash
pnpm install
```

## Usage

```bash
# start dev server
pnpm dev

# build for production
pnpm build

# test the prod build
pnpm preview
```

You can check other scripts in `package.json` of the root repo and each package
